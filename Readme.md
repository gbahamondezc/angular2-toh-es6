### angular2-toh-es6

[Angular 2 Tour of Heroes](https://angular.io/docs/ts/latest/tutorial/toh-pt1.html), using ES6 + traceur,  instead of Typescript.

##### Run project

```bash
$ git clone git@github.com:gbahamondezc/angular2-toh-es6.git

$ cd angular2-toh-es6

$ npm install -g gulp-cli
$ npm install

$ gulp serve
```
